# Project template using laravel breeze UI with localization

## Setup (these steps were not tested, this is just a guide)

1. Clone this repo
2. Install dependencies
```
composer install
npm install
npm run dev
```
3. Add database credentials to `.env` file.
4. Migrations 
```
php artisan migrate
```
